# Tic-Tac-Toe Web Application

## Task
Web Development Internship – Task 03

## Description
A Tic-Tac-Toe web application developed using HTML, CSS, and JavaScript.
The game handles user clicks, tracks the game state, checks for winning
conditions, and provides sound effects for clicks and winning moments,
making the gameplay interactive and engaging.

## Features
- Two-player Tic-Tac-Toe gameplay
- Click sound effect for player moves
- Winning sound effect on game completion
- Game state tracking and win detection
- Reset and replay functionality
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
